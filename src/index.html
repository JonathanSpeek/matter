A boilerplate for SpaceGrid using PostCSS & Webpack

Features

The best fucking shit, ever.

Spacegrid

A basic, responsive grid layout to help you get started on your next project. View Project

PostCSS

PostCSS plugins targeting consistency with the CSS4 specification.

Optimization & Minification

CSS & Image optimization/minification with cssnano and image-min.

Additional Features

Matter includes many other features for easy, fast local development including:

  • Webpack 2
  • ES2015
  • Browsersync
  • Minification
  • Sourcemapping

Getting Started

Get the app running for local developement.

Clone the repo 
                  git clone https://github.com/wad3g/matter.git
cd matter
Install dependencies with yarn or npm 
                        yarn install
                        npm i
Start for local developing and begin watching files for compilation 
                  yarn start
